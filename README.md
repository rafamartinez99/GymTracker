# GymTracker

Aplicación Android para registrar ejercicios de gimnasio y seguimiento de pesos máximos.

## Stack Técnico
- **Kotlin 2.0** con Jetpack Compose
- **Room** para base de datos local
- **Firebase Firestore** para sincronización
- **Hilt** para inyección de dependencias
- **Vico** para gráficas de progreso

## Características
- 📋 Lista de ejercicios por categoría (piernas, pecho, espalda, hombros, brazos)
- 💪 Registrar pesos levantados
- 📊 Gráficas de progreso
- 🔄 Sincronización en la nube
